# LoRa-Based Multi-Hop Predictive Energy Demand Modeling for Wireless Communication Networks

## 📖 Project Overview
This project focuses on predicting energy demand in wireless communication networks using a LoRa-based multi-hop architecture. The system uses machine learning techniques to analyze network behavior and predict energy requirements efficiently.

## 🎯 Objective
- To design a multi-hop LoRa communication system
- To analyze energy consumption in wireless sensor networks
- To predict energy demand using machine learning models
- To improve efficiency and reliability in IoT communication

## ⚙️ Technologies Used
- Python
- Machine Learning
- Jupyter Notebook
- Pandas & NumPy
- Scikit-learn
- LoRa Communication Concepts

## 📊 Models Used
- Random Forest Classifier
- Support Vector Machine (SVM)
